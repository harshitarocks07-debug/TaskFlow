# TaskFlow - Software Specification Document

## Project Overview

TaskFlow is a modern To-Do application developed using React and Vite. The application helps users organize and manage tasks efficiently through a clean and visually appealing interface.

The project focuses on:
- productivity
- simplicity
- modern UI design
- easy task management

---

# Objectives

The main objectives of TaskFlow are:

- Allow users to manage daily tasks
- Improve task organization
- Provide priority-based task management
- Track task completion status
- Create an attractive and responsive UI

---

# Features

## Core Features

### Add Tasks
Users can create new tasks using the task input field.

### Delete Tasks
Users can remove tasks from the dashboard.

### Complete Tasks
Users can mark tasks as completed.

### Undo Completion
Users can undo completed tasks.

---

# Advanced Features

## Priority Management
Users can assign:
- High priority
- Medium priority
- Low priority

to each task.

---

## Due Date Selection
Users can assign due dates to tasks using the calendar input field.

---

## Modern UI Design
The application includes:
- pink gradient background
- responsive layout
- rounded task cards
- styled buttons
- clean typography

---

# Technologies Used

- React

---

# Functional Requirements

The system should:

- allow task creation
- allow task deletion
- allow task completion
- allow undo functionality
- allow priority assignment
- allow due date assignment

---

# Non-Functional Requirements

The application should:
- be responsive
- be easy to use
- have a clean UI
- provide smooth interaction
- support modern browsers

---

# User Workflow

1. User enters a task
2. User selects priority level
3. User selects due date
4. User clicks Add Task
5. Task appears in dashboard
6. User manages tasks using:
   - Complete button
   - Undo button
   - Delete button
